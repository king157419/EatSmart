# 个性化营养目标计算器使用指南

## 功能介绍

根据用户的身高、体重、年龄、性别、活动水平和健康状况，自动计算个性化的每日营养目标。

## 访问方式

1. **从主页访问**：点击顶部导航栏的 "🧮 营养计算" 按钮
2. **直接访问**：浏览器打开 `http://localhost:3000/calculator`

## 使用步骤

### 1. 输入基本信息

- **体重（公斤）**：例如 70
- **身高（厘米）**：例如 170
- **年龄**：例如 65
- **性别**：男性 / 女性

### 2. 选择活动水平

- **久坐（很少运动）**：办公室工作，很少运动
- **轻度活动（每周1-3天）**：每周1-3天轻度运动
- **中度活动（每周3-5天）**：每周3-5天中等强度运动
- **高度活动（每周6-7天）**：每周6-7天高强度运动
- **非常活跃（每天高强度运动）**：每天高强度运动或体力劳动

### 3. 选择健康状况

- ☑️ **有糖尿病**：勾选后会控制碳水化合物摄入（150-250g/天）
- ☑️ **有胰腺炎病史**：勾选后会严格限制脂肪摄入（< 30g/天）

### 4. 计算营养目标

点击 **"🧮 计算营养目标"** 按钮，系统会显示：

#### BMI 分析
- BMI 值
- 体重状态（偏瘦/正常/超重/肥胖）
- 健康建议

#### 个性化营养目标
- **热量**（千卡/天）
- **蛋白质**（克/天）
- **脂肪**（克/天）⚠️ 胰腺炎患者严格限制
- **碳水化合物**（克/天）
- **膳食纤维**（克/天）

### 5. 应用到系统

点击 **"✅ 应用到系统"** 按钮，计算出的营养目标会自动更新到数据库，成为新的每日营养目标。

## 计算原理

### 基础代谢率（BMR）

使用 **Mifflin-St Jeor 公式**：

- **男性**：BMR = 10 × 体重(kg) + 6.25 × 身高(cm) - 5 × 年龄 + 5
- **女性**：BMR = 10 × 体重(kg) + 6.25 × 身高(cm) - 5 × 年龄 - 161

### 每日总能量消耗（TDEE）

TDEE = BMR × 活动系数

| 活动水平 | 系数 |
|---------|------|
| 久坐 | 1.2 |
| 轻度活动 | 1.375 |
| 中度活动 | 1.55 |
| 高度活动 | 1.725 |
| 非常活跃 | 1.9 |

### 营养素分配

#### 蛋白质
- 推荐：1.0-1.2 g/kg 体重
- 老年人需要更多蛋白质维持肌肉

#### 脂肪
- **胰腺炎患者**：严格限制 < 30g/天（硬性限制）
- **正常人**：20-30% 热量来自脂肪

#### 碳水化合物
- **糖尿病患者**：45-60% 热量，上限 250g/天
- **正常人**：50-60% 热量
- 最低 150g/天（大脑需求）

#### 膳食纤维
- 推荐：25-30g/天
- 有助于血糖控制和消化健康

## 示例

### 示例 1：65岁男性，糖尿病+胰腺炎

**输入**：
- 体重：70kg
- 身高：170cm
- 年龄：65岁
- 性别：男性
- 活动水平：轻度活动
- 健康状况：糖尿病 ✓、胰腺炎 ✓

**输出**：
- 热量：1785 千卡
- 蛋白质：77 克
- 脂肪：25 克 ⚠️（胰腺炎限制）
- 碳水化合物：250 克
- 膳食纤维：25 克
- BMI：24.2（超重）

### 示例 2：60岁女性，仅糖尿病

**输入**：
- 体重：60kg
- 身高：160cm
- 年龄：60岁
- 性别：女性
- 活动水平：轻度活动
- 健康状况：糖尿病 ✓、胰腺炎 ✗

**输出**：
- 热量：1520 千卡
- 蛋白质：66 克
- 脂肪：42 克（正常范围）
- 碳水化合物：210 克
- 膳食纤维：25 克
- BMI：23.4（正常）

## 注意事项

1. **首次使用**：建议根据父亲的实际情况填写信息
2. **定期更新**：体重变化时重新计算
3. **医生建议**：计算结果仅供参考，请遵医嘱
4. **胰腺炎限制**：脂肪摄入是硬性限制，必须严格遵守
5. **血糖监测**：糖尿病患者需配合血糖监测调整饮食

## API 接口

### 计算营养目标

```bash
POST /api/nutrition/calculate
Content-Type: application/json

{
  "weight_kg": 70,
  "height_cm": 170,
  "age": 65,
  "gender": "male",
  "activity_level": "light",
  "has_diabetes": true,
  "has_pancreatitis": true
}
```

### 计算并应用

```bash
POST /api/nutrition/apply-calculated
Content-Type: application/json

{
  "weight_kg": 70,
  "height_cm": 170,
  "age": 65,
  "gender": "male",
  "activity_level": "light",
  "has_diabetes": true,
  "has_pancreatitis": true
}
```
