# 糖尿病 + 胰腺炎康复期 综合禁忌食物清单

> 本清单综合了 2 型糖尿病和急性胰腺炎康复期的饮食限制。
> 由于患者同时患有两种疾病，饮食限制比单一疾病更严格。

---

## ⛔ 绝对禁忌（任何情况下都不能吃/喝）

### 酒精类
- **所有含酒精饮品**：白酒、啤酒、红酒、黄酒、鸡尾酒
- 含酒精食品：酒酿、醪糟、酒心巧克力
- 料酒烹饪的菜品（如需去腥，用姜片/柠檬汁替代）
- **理由**：酒精直接损伤胰腺细胞，同时干扰降糖药效果

### 动物内脏
- 猪肝、猪肾、猪脑
- 鸡肝、鸭肝
- 肥肠
- **理由**：高胆固醇 + 高脂肪 + 高嘌呤，加重胰腺负担

### 油炸食品
- 油条、油饼
- 炸鸡、炸鱼
- 薯条、薯片
- 油炸糕点（麻花、馓子等）
- **理由**：高脂肪（胰腺炎禁忌）+ 高GI（糖尿病禁忌）

---

## 🔴 严格限制（极少量或避免）

### 高脂肪食物
| 食物 | 每日限制 | 原因 |
|:---|:---|:---|
| 肥肉/五花肉 | 避免 | 脂肪含量极高 |
| 排骨 | 避免或去脂 | 脂肪含量高 |
| 肉汤/骨汤 | 避免 | 含大量隐形脂肪 |
| 奶油/黄油 | 避免 | 纯脂肪 |
| 猪油/牛油 | 避免 | 饱和脂肪酸 |
| 蛋黄 | ≤1个/日 | 胆固醇较高 |
| 蟹黄/鱼籽 | 避免 | 高胆固醇高脂肪 |
| 沙拉酱/蛋黄酱 | 避免 | 隐形脂肪大户 |
| 花生酱 | 极少量 | 脂肪含量高 |

### 高糖食物
| 食物 | 限制 | 原因 |
|:---|:---|:---|
| 蛋糕/甜点 | 避免 | 高糖 + 高脂双重打击 |
| 糖果 | 避免 | 纯糖 |
| 冰淇淋 | 避免 | 高糖 + 高脂 |
| 巧克力 | 避免 | 高糖 + 高脂 |
| 蜜饯/果脯 | 避免 | 浓缩糖分 |
| 蜂蜜 | 避免 | GI = 73 |
| 果酱 | 避免 | 高糖 |

### 高糖饮料
| 饮品 | 限制 | 原因 |
|:---|:---|:---|
| 可乐/雪碧 | 禁止 | 大量游离糖 |
| 奶茶 | 禁止 | 高糖 + 反式脂肪 |
| 果汁（包装/鲜榨） | 避免 | 缺少纤维，糖分吸收快 |
| 运动饮料 | 避免 | 含糖 |
| 甜豆浆 | 避免 | 额外添加糖 |

### 高GI主食
| 食物 | 限制 | 原因 |
|:---|:---|:---|
| 白粥/稀饭 | 避免 | GI > 90，血糖飙升 |
| 糯米/粽子 | 避免 | GI = 87 |
| 年糕 | 避免 | 高GI |

---

## 🟡 需谨慎食用（可以吃但要注意量和方式）

| 食物 | 怎么吃 | 注意 |
|:---|:---|:---|
| 白米饭 | 每餐 ≤ 1小碗，搭配蔬菜 | 掺糙米/燕麦降低GI |
| 馒头 | 选杂粮馒头，1个以内 | 白馒头 GI = 88 |
| 面条 | 选荞麦面/全麦面 | 煮软后GI更高 |
| 水果 | 每日150-200g，餐间吃 | 选低GI水果 |
| 西瓜 | 极少量（50-100g） | GI = 72 |
| 香蕉 | 半根 | GI = 55，含糖高 |
| 坚果 | 每日一小把（15-25g） | 脂肪含量高 |
| 牛奶 | 选低脂/脱脂 | 全脂奶脂肪偏高 |
| 食用油 | 每日 ≤ 15g（约3茶匙） | 正常人25g，胰腺炎更少 |

---

## 🟢 安全食物列表（放心吃）

### 蔬菜（每日 500g+）
菠菜、西兰花、冬瓜、黄瓜、苦瓜、番茄、芹菜、莴苣、油麦菜、白菜、生菜、青椒、花椰菜、香菇、木耳、金针菇、海带

### 低脂蛋白质
去皮鸡胸肉、鱼肉（清蒸）、虾仁、豆腐、蛋清

### 低GI主食
燕麦、荞麦面、全麦面包、糙米、杂粮粥、绿豆

### 低GI水果
苹果、梨、柚子、草莓、樱桃、猕猴桃

### 调味品
醋、少量酱油、姜片、葱白、少量盐

---

## 隐形危险食物（容易被忽视）

以下食物看似"健康"，实则对糖尿病+胰腺炎患者有风险：

| 食物 | 隐藏风险 |
|:---|:---|
| **鸡汤/骨汤** | 表面清淡，实则脂肪含量很高 |
| **全脂酸奶** | "酸奶"听起来健康，但全脂版脂肪高 |
| **果蔬汁** | 缺少纤维，糖分快速吸收 |
| **无糖饼干** | "无糖"但淀粉含量高，升糖快 |
| **香肠/培根** | 加工肉类，高脂肪高钠 |
| **千层饼/葱油饼** | 面食+大量油脂 |
| **咖喱/麻辣烫** | 汤底含大量油脂 |
| **沙拉** | 沙拉酱脂肪极高 |
| **瘦肉月饼** | 高脂高糖 |
